    searchForContent

    public any searchForContent(string query, string file_extensions='', string type='', numeric limit='', string asUserID='[runtime expression]')

    Parameters:
    query - The string to search for. Box matches the search string against object names, descriptions, text contents of files, and other data.
    file_extensions - Limit searches to specific file extensions like [pdf], [png], or [doc]. The value can be a single file extension or a comma-delimited list of extensions. For example: [png,md,pdf]
    type - The type of objects you want to include in the search results. The type can be [file], [folder], or [web_link].
    limit - The maximum number of items to return. The default is 30 and the maximum is 200.
    asUserID - BoxID of user to perform action on behalf of.
